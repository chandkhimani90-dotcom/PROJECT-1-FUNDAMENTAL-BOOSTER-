# 🚀 Fundamental Booster – Interactive Personal Data Collector

## 📖 Project Overview
This project is developed in Python as part of the **Fundamental Booster** assignment. It is an interactive Python application that collects personal information from the user, performs basic calculations, demonstrates type casting, and displays the data type and memory address of variables.

---

## 🎯 Objective
The objective of this project is to demonstrate the use of fundamental Python concepts, including:

- `print()` function
- `input()` function
- Variables
- Data Types
- Arithmetic Operators
- Type Casting
- Built-in Functions (`type()` and `id()`)

---

## ✨ Features

- Collects user information:
  - Name
  - Age
  - Height (in cm)
  - Favourite Number
- Calculates the estimated birth year.
- Converts height from centimeters to meters.
- Demonstrates type casting.
- Performs arithmetic operations.
- Displays the data type using `type()`.
- Displays the memory address using `id()`.
- Provides a complete project summary.
